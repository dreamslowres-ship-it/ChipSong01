Comandos básicos:
  a b c d e f g   → Notas (la, si, do, re, mi, fa, sol)
  + / -           → Sostenido / Bemol
  o<n>            → Octava (o4, o5...)
  > / <           → Subir/Bajar octava
  l<n>            → Duración (1=redonda, 4=negra, 8=corchea)
  .               → Puntillo
  r               → Silencio
  v<n>            → Volumen (0-15)
  @<n>            → Instrumento
  t<n>            → Tempo (BPM)
  [ ... ]<n>      → Repetir n veces
  $nombre         → Llamar macro
  E<n>,<params>   → Efectos
  #               → Comentario

Efectos:
  E0,x           → Vibrato
  E1,x,y         → Portamento
  E2,x           → Duty cycle
  E3,t,a,d,s,r   → Envolvente ADSR
  E4,x           → Slide volumen
  E5             → Glissando

Directivas:
  #chip nes|gb|c64|genesis
  #tempo <BPM>
  #loop <nº>
  #master_volume <0-15>
  #transpose <semitones>
